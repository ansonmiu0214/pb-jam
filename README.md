# PB Jam - Spotify Race Playlist Visualizer

## Overview

PB Jam is a TypeScript-based web application that helps runners and athletes visualize and manage their race playlists. It integrates with Spotify to allow users to:

- Create race/pace plans with custom splits
- Link Spotify playlists to pace plans
- Visualize track duration against race splits on an interactive timeline
- Reorder playlist tracks to match race pacing

## Project Status

This project is in initial setup phase. Core dependencies and configuration are being established for Tier 1 (MVP) development.

## Tech Stack

- **Language:** TypeScript
- **Module System:** ES Modules
- **Build/Dev Tools:** npm scripts
- **Testing:** Vitest
- **Linting:** ESLint
- **Backend Services:** Firebase (Firestore, Auth), Spotify API
